# CLAUDE.md

Este projeto centraliza instruções para agentes de IA em dois arquivos na raiz — leia os dois antes de começar qualquer tarefa:

1. **[`HANDOFF.md`](./HANDOFF.md)** — estado atual do projeto: os dois ambientes de produção em uso, o que já foi construído, pendências conhecidas. Leia primeiro.
2. **[`AGENTS.md`](./AGENTS.md)** — convenções de código, padrões de arquitetura, runbook de deploy, armadilhas já encontradas.

Não duplique o conteúdo desses arquivos aqui. Se aprender algo novo sobre o projeto durante uma sessão que valha a pena persistir, atualize o arquivo correto (estado → `HANDOFF.md`, convenção → `AGENTS.md`) em vez de criar um terceiro lugar.
